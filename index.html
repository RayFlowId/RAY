Pengenalan HTML

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. HTML digunakan untuk mengatur konten di web.

Struktur Dasar HTML

        <!DOCTYPE html>
        <html>
            <head>
                <title>Judul Halaman</title>
            </head>
            <body>
                <h1>Halo, Dunia!</h1>
                <p>Ini adalah paragraf pertama saya.</p>
            </body>
        </html>

Elemen Penting dalam HTML

  • <h1> - <h6>: Judul
  • <p>: Paragraf
  • <a>: Tautan
  • <img>: Gambar
